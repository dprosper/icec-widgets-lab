## Running locally

You need Ruby and gem before starting, by default these are installed on Mac OS, then:

```bash
# install bundler
gem install bundler

# clone the project
git clone hhttps://github.com/dprosper/icec-widgets-lab.git
cd icec-widgets-lab

# run jekyll with dependencies
bundle exec jekyll serve
```
